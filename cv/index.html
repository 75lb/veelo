  • Nature provides a base class (Thing) to enable more accurate classification of the things (DatabaseConfig, VideoFileList, YoungMan, FlammableCar etc.) in your world (function or app). These Things can be reused, subclassed or merged with other Things to create new hybrids. The Thing class, when subclassed/instantiated adds useful features to each object:
    • Fine-grained type and value validation per property
    • Direct command line, environment, external file or object literal loading capability
    • Merging and cloning
    • property grouping and filtering
    • methods for duck typing
  • Got a thousand jobs to do? Get an execution plan with Work. Create one of more instances of Job, specifying the name, command, arguments and whether the job may run in parallel to others. Add each Job to a Queue for execution in the order provided. Each Job can have a sub-Queue to be executed onSuccess, onProgress or onFailure, each of which containing its own set of Job instances. And so the hierarchy grows! Work can also project a dashboard, providing real-time progress information on any standard Node.js Stream.

Approach

  • Methodical. Agile Unified Process or Scrum preferred. Don't like 'headless chicken'.
